Description

Hi everyone. It's been quite a while from the last time I've uploaded something here.

Recently I've been working on a Warcraft 3 map, that has a loot drop system. My goal was to make every dropable item to have a different model, so the player could instantly recognise what item he/she has found. Of course, I needed lots of item models for that.

Most of the item models you can find for Warcraft are pretty low quality, so I decided to create some better quality items by myself.

I thought these models may be pretty useful, so I'm sharing them with you.

The pack contains 30 item models, 8 of them have some colour variations.
All the items use Warcraft 3 icons as textures.
Potions and keys also feature some custom variations made by me, and they obviously use custom icons.
