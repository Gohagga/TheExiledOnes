Note about model:
ZDinsFire, ZFaroresWind and ZNayrusLove continue on the huge Zelda kick im on. I got these out 2 days after my Light Arrow model, thats damn quick. They all use the same base model, but use different skins (which is how i got them out so quick hrhrhrhrhrhr). In the future Im going to try my hand at making all 3 models into one package, and then the user can have the World Editor separate the three models out by using the "Art - Required Animation Name" function (exactly like the human scout tower, if youre familiar with how that model works). That way people who need to use all 3 models will save a couple KB for their map; every KB counts! The animations for these models are fairly unremarkable, they dont rotate, sparkle or move. I dont think i would have liked the rotation look, and any sparkling would have taken away from the emitters inside the crystals.
A future project might be the spell models for the acctual spells that these items were used for but, I dunno, I dont like doing spell models very much. As of right now I have no next project. I might take a break.
